## Project Overview
This project analyzes customer spending behavior and membership tiers using SQL.  
The goal is to understand which customer groups generate the most revenue and identify opportunities for retention strategies.

## Tools Used
- SQL
- PostgreSQL
- pgAdmin
- GitHub

## Key Insights
- Total revenue generated: $295,883.60
- Gold members generate the highest revenue.
- Gold members also spend significantly more per customer than Silver or Bronze members.
